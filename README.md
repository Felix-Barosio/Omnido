## Installation & Set up

Clone the repository

```
https://github.com/Felix-Barosio/Omnido
```

Install dependecies

```
bundle install
```

```
rails db:create
```

```
rails db:seed
```

```
npm install --prefix client
```

Run

```
rails s
```

```
npm start --prefix client
```
